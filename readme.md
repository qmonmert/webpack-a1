# Webpack

### Install

* npm install webpack -g
* npm install webpack-dev-server -g
* npm install
* webpack

### Run

* webpack-dev-server --content-base app/ --hot

or

* npm start

### Visualize

* http://localhost:8080/webpack-dev-server/
* http://localhost:8080/
